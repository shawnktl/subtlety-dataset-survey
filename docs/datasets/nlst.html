NLST (National Lung Screening Trial)

  • Tier: C — adjacent; high-value for a custom missed-on-prior subset but not turnkey
  • Modality / anatomy: Low-dose chest CT / lung cancer screening
  • Finding type: Lung cancer (screening outcomes)

Subtlety / perceptibility label

No native "missed-on-prior" or "perceptibility" labeling. The "missed cancer" subset (cancers detected at a later screening round whose prior CTs retrospectively showed the lesion) has been characterized in published work (PMC7759925) — ~74 missed cancers identifiable across ~113 prior CTs — but this is a derivable subset, not a packaged dataset with missed-on-prior labels.

Label provenance

Standard NLST outcomes (screen result, cancer dx).

Size

~54k participants, 75k+ CT exams.

Open via NCI CDAS with DUA. https://cdas.cancer.gov/nlst/
