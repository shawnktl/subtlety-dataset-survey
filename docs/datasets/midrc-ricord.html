MIDRC / RICORD

  • Tier: B — reader-disagreement / multi-rater design (difficulty proxy)
  • Modality / anatomy: CXR (+ CT) / COVID severity
  • Finding type: COVID severity scoring (modified RALE)

Subtlety / perceptibility label

No perceptibility rating per se; reader divergence per image behaves as a difficulty proxy. Modified RALE scoring scheme.

Label provenance

Each image read by 3 physicians for COVID severity; majority/average used as label. Reader divergence per image is preserved as the labeling input.

Size

Tens of thousands of COVID cases across MIDRC contributing institutions.

Access / licensing

Open via the MIDRC portal (registration + DUA). https://www.midrc.org/

Caveats

  • Severity scoring is closer to a clinical-condition rating than a perceptibility rating — but reader divergence behaves as a difficulty proxy.

Relevance to detectability

Multi-reader severity scoring with explicit per-reader divergence available in some subsets.
