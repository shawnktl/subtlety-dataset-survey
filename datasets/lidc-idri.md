# LIDC-IDRI

- **Tier:** A — direct subtlety/perceptibility/conspicuity label
- **Modality / anatomy:** Thoracic CT / pulmonary nodules
- **Finding type:** Lung nodules (≥3 mm)

## Subtlety / perceptibility label
Per-nodule **subtlety rating, scale 1 ("extremely subtle") to 5 ("obvious")**, annotated independently by up to 4 thoracic radiologists. Plus 7 other characterization ratings (margin, lobulation, spiculation, texture, etc.).

## Label provenance
Independent ratings by up to 4 thoracic radiologists per nodule (per-reader signal preserved).

## Size
1,018 cases, ~2,600+ nodules with characterization ratings.

## Access / licensing
Open via TCIA. https://www.cancerimagingarchive.net/collection/lidc-idri/

## Caveats
- Already in use in this project. Listed for completeness.

## Relevance to detectability
Direct subtlety label — the project's current/anchor source.
