# PadChest

- **Tier:** C — adjacent; label-noise study object more than a subtlety dataset
- **Modality / anatomy:** CXR / 174 findings, hierarchical taxonomy
- **Finding type:** Thoracic findings (174-finding taxonomy)

## Subtlety / perceptibility label
**Binary presence/absence — no uncertainty or subtlety axis.**

## Label provenance
27% manually annotated by trained physicians; remainder by RNN trained on Spanish reports.

## Size
160k+ images, 67k patients.

## Access / licensing
Open with registration. http://bimcv.cipf.es/bimcv-projects/padchest/

## Caveats
- Binary presence/absence — no uncertainty or subtlety axis.

## Relevance to detectability
Adjacent; useful as a label-noise study object more than a subtlety dataset.
