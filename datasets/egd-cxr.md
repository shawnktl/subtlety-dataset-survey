# EGD-CXR (Eye Gaze Data for Chest X-rays)

- **Tier:** C — process-level (eye-tracking) signal, single reader
- **Modality / anatomy:** CXR / eye-gaze + dictation
- **Finding type:** Thoracic (with gaze + report)

## Subtlety / perceptibility label
No subtlety label; gaze/dwell features as conspicuity proxies.

## Label provenance
1,083 CXRs read by a **single radiologist** with eye-tracker + audio dictation + transcribed report.

## Size
1,083 CXRs.

## Access / licensing
Open via PhysioNet. https://physionet.org/content/egd-cxr/1.0.0/

## Caveats
- Single reader — no inter-rater signal.
- Smaller than REFLACX.

## Relevance to detectability
Useful for gaze/dwell features as conspicuity proxies but limited.
