# SIIM-ACR Pneumothorax

- **Tier:** C — adjacent only
- **Modality / anatomy:** CXR / pneumothorax
- **Finding type:** Pneumothorax (pixel-level masks)

## Subtlety / perceptibility label
**No explicit subtlety/conspicuity label** in the published dataset. Pneumothorax detection is a classic perceptual-difficulty domain (apical/small PTX often missed), but difficulty has to be inferred from mask size / location, not labeled.
